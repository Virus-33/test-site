# Кодекс чести

Данный кодекс является действующими правилами для проведения ПВП (боёв 1на1 или более, проводимых исключительно между игроками вне рейдов)

[**Награда** за бой по чести: 100 мк]

1. **Дай шанс в первые секунды**
>_Нельзя использовать более одной способности, наносящей урон в первый раунд_
2. **Лишь собственная воля несёт честную победу**
>_Подчинение работает как стан. Стан работает не более одного хода_
3. **Сегодняшний враг может стать завтрашним другом**
>_Если любая из сторон сдалась, то она тут же полностью выходит из боя_
4. **Уважай соперника своего. Он — не добыча**
>_В один раунд может сработать до двух ловушек траппера_
5. **Этот бой — лишь возможность для славы, а не критическая ситуация**
>_Станокаст псайкеров запрещён_
6. **Держи себя в руках в честно́м бою**
>_Если на одной из сторон больше участников чем на другой, то ходить в раунд может лишь **случайное** количество существ
превосходящей стороны, но **не меньше** чем количество другой команды
(мёртвые персонажи при подсчёте участников учитываются, но не учитываются мёртвые призванные существа)_
